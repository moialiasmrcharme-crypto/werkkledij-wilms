# Werkkledij Wilms

Volledige prototypeversie met:
- werknemerweergave
- beheerderweergave
- Excel-import van werkblad INVOER
- open aanvragen
- goedkeuren / weigeren

## Lokaal starten
1. npm install
2. npm run dev

## Online zetten
1. upload deze volledige map naar GitHub
2. koppel de repo aan Vercel
3. deploy
