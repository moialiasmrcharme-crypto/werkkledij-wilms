Werkkledijportaal - React/PWA prototype

Inhoud:
- React webapp met werknemer- en beheerderweergave
- Excel import van werkblad INVOER
- PWA manifest zodat de app later op iPhone aan het beginscherm kan worden toegevoegd

Lokaal starten:
1. Installeer Node.js
2. Open terminal in deze map
3. npm install
4. npm run dev

Build maken:
- npm run build

Online zetten:
- deploy de inhoud via Vercel, Netlify, of een eigen webserver
- open daarna de URL in Safari op iPhone
- kies Deel > Zet op beginscherm

Belangrijk:
- dit is een prototype zonder echte login of databank
- nieuwe bestellingen worden enkel in het browsergeheugen bijgehouden
- voor productie moet er nog een backend en authenticatie bijkomen
